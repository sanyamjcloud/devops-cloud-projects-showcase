# DevOps & Cloud Projects Showcase

## Overview

This repository contains a curated collection of hands-on projects focused on Cloud Computing, DevOps practices, and system automation.

The projects demonstrate practical implementation of deployment pipelines, containerization, cloud infrastructure setup, and Linux-based automation.

## Project Categories

### Cloud Engineering

* AWS EC2 instance setup and configuration
* EBS volume management
* S3 region configuration and validation
* EC2 launch template deployment

### DevOps & CI/CD

* Flask application CI/CD pipeline with Docker
* Automated deployment using GitHub Actions

### Monitoring & Reliability

* System monitoring and logging setup
* Self-healing infrastructure using Docker and Nginx
